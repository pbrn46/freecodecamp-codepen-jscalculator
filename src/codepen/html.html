FreeCodeCamp Javascript Calculator Challenge

It works better than the demo in the challenge!

Note that these traditional style calculators do not follow "order of operations". The formulae the list are desscribed as such (from left to right). For example, the formula 1 + 2 * 3 / 4 is actually (((1 + 2) * 3) / 4), equalling 13.5 rather than 11.

Features

  • Override previous operation (ie. pressing +, then - will enter subtract mode)
  • Keyboard keys: Standard keys, as well as Escape for AC, and Backspace for CE
  • Internal: Math operations don't rely on history for extra fast (although unnoticeable) calculations! Only CE relies on formula history so we can cancel multiple operations.
  • After writing a formula, keep pressing =!
  • To start a new formula, hit equals to finalize current formula. Then, type a number.
